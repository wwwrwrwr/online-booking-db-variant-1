## Детальное описание дефектов

### Дефект 1 - Пагинация не сохраняет поиск
**Серьёзность:** Значительный
**Шаги воспроизведения:**
1. Открыть список клиентов
2. Ввести фамилию в поле поиска, нажать Найти
3. Нажать на страницу 2 в пагинации

**Ожидаемый результат:** Список фильтруется по фамилии на странице 2
**Фактический результат:** Поиск сбрасывается, показываются все клиенты
**Причина:** В ссылках пагинации в list.php не передаётся параметр search

### Дефект 2 - Нет отступов между кнопками действий
**Серьёзность:** Косметический
**Описание:** Кнопки Просмотр, Изменить и Удалить расположены вплотную друг к другу. На мобильном устройстве легко нажать не ту кнопку.

## Рекомендации
- Добавить параметр search в ссылки пагинации
- Добавить отступы между кнопками действий
