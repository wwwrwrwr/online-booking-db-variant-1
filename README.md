2. Выполнить sql/schema_variant.sql

## 🛠 Уровень доступа к данным (DAL)

Реализован объектно-ориентированный слой работы с БД на PHP 8+ с использованием PDO. Архитектура построена на паттерне Repository с абстрактным базовым классом и конкретными репозиториями для каждой таблицы схемы.

### Структура
- `src/Database.php` — Singleton-подключение к PDO с настройками безопасности
- `src/AbstractRepository.php` — базовые CRUD-методы с поддержкой фильтрации, сортировки и лимитов
- `src/*Repository.php` — специализированные классы для таблиц `clients`, `services`, `appointments`, `medical_cards`
- `demo.php` — демонстрационный скрипт последовательного вызова всех операций

###  Запуск
1. Скопируйте `config.example.php` в `config.php` и укажите параметры подключения
2. Убедитесь, что база данных создана и содержит таблицы из файла `schema.sql`
3. Запустите демонстрацию: `php demo.php` (или откройте в браузере, если используется веб-сервер)

### Безопасность
- Все запросы используют подготовленные выражения PDO (`prepare`/`execute`)
- Параметры сортировки проходят валидацию по белому списку (`ASC`/`DESC`)
- Ошибки целостности БД перехватываются и преобразуются в `RepositoryException`
- Пароли и конфигурация вынесены в `.gitignore`
